# xi-bundle-gl2

GL(2) extension of the xi-bundle framework to elliptic curve L-functions.

## Overview

This repository contains the specialized GL(2) results from the xi-bundle research programme:
- Elliptic curve L-functions (11a1, 37a1)
- Ramanujan Delta function (weight 12)
- sigma-uniqueness bifurcation mechanism
- Connection to the Birch and Swinnerton-Dyer (BSD) conjecture
